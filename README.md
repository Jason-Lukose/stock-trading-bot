# Stock Trading Bot

Paper-trading research bot using Alpaca.

This project is for educational and research purposes only. It defaults to paper trading and should not be used with live funds until strategies have been backtested, paper traded, reviewed, and risk-tested.

## Structure

- `/docs` — architecture, strategy specs, risk rules, and build plan
- `/bot` — trading bot source code
- `/tests` — automated tests
- `/data` — local market data storage
- `/reports` — backtest and paper trading reports
- `/reference` — source PDFs or notes
